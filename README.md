# 收集汇总个人常用的 skill


## 开源

- [comet](https://github.com/rpamis/comet/blob/master/README-zh.md), 它用统一的跨平台运行时把 OpenSpec 产物、Superpowers 执行方法论、Skill 创建、评估与发布串成一条工作闭环

- [superpowers](https://github.com/obra/superpowers)，是一种完整的软件开发方法论，用于指导你的编程智能体，它基于一系列可组合的技能和一些初始指令，以确保这些技能被智能体正确使用。

- [openspec](https://github.com/Fission-AI/openspec), AI 编程助手中的规范驱动开发（SDD）方法。



## 个人
